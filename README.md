# Soft Deepslate
A Minecraft MOD that adjusts the mining speed of Deepslate to be the same as that of stone.  
Target blocks and speed can be changed by config.  

深層岩を石と同等の採掘速度に調整するMinecraftのMOD。  
対象となるブロックや速さはコンフィグで変更可能。
